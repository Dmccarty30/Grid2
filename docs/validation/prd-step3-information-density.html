Line 316 — Hedging
"~8% of male field workers may be color-blind."

Assessment: "May be" introduces uncertainty where established medical data exists. Approximately 8% of males are affected by color vision deficiency — this is a well-documented statistic.

Suggested fix: "~8% of male field workers have some form of color-vision deficiency."

Line 524 — Passive Voice
"Contractor PII (W-9 documents, insurance documents) is stored in a secured storage bucket with restricted access — only Super Admin can view."

Assessment: Passive construction ("is stored in") obscures the actor. However, this is an NFR describing a system storage property, so passive voice is marginally acceptable. A more direct form strengthens the requirement.

Suggested fix: "NFR8: Supabase stores contractor PII (W-9, insurance) in a secured storage bucket — only Super Admin can access."

L231: "Form structure and entry process may vary by utility"

These are not true violations but are noted for completeness. Suggested fix: No change required. These are legitimate uses of "may" to express possibility in external conditions. If tighter language is desired: "utility contracts can specify..." or "form structure varies by utility."
