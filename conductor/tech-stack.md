# Technology Stack - Grid Electric Services Platform

## Frontend Framework
- **Next.js (App Router):** Leveraging React 19 and Next.js 16 for high-performance server-side rendering, robust routing, and a seamless developer experience.

## Language
- **TypeScript:** Ensuring end-to-end type safety, improving maintainability, and providing a superior developer experience through static analysis.

## UI & Styling
- **Tailwind CSS (v4):** A utility-first CSS framework for rapid UI development and consistent design implementation.
- **shadcn/ui (Radix UI):** A collection of accessible, high-quality UI components built with Radix UI and styled with Tailwind CSS.

## State Management
- **Zustand:** A lightweight, high-performance state management library for managing global application state.
- **TanStack Query (React Query):** Handling server state, caching, and data synchronization with ease.

## Backend & Infrastructure
- **Supabase (BaaS):** Utilizing Supabase for Authentication, PostgreSQL database with Row-Level Security (RLS), and Storage for files and photos.

## Specialized Capabilities
- **Dexie.js (IndexedDB):** Providing a robust offline storage solution to ensure data persistence during field operations in remote areas.
- **Mapbox GL JS:** Integrating high-performance maps and location services for ticket routing and field assessments.
